CSE4153/CSE6153 Data Communication and Computer Networks Syllabus

Introduction

  • Overview of computer networks, hardware, and software
  • OSI and TCP reference models
  • Example networks and data communication services
  • Data representation, alignment, Big- vs Little-Endian

Physical and Data Link Layers

  • Theoretical basis for data communication; transmission media and systems
  • Data link service; framing, error and flow control; elementary protocols
  • Sliding window protocols and flow control
  • Medium access sublayer; channel allocation, collision, ALOHA, CSMA, CSMA/CD;; CSMA-CA, Ethernet and Wireless Ethernet standards;

Network Layer, Internetworking, and IP

  • Network layer service; internal organization; best-effort delivery; connections, virtual circuits, datagrams
  • Routing, routing algorithms; ARP, DHCP; congestion control algorithms
  • Internetworking; network differences; datagram fragmentation
  • The Internet; IP protocol, IP addresses, Internet control protocols; IP version 6 (IPv6)

    Transport Layer

    • Service; guaranteed delivery, stream-oriented; quality of service; primitives
    • Transport protocols; addressing, connections; flow control and buffering
    • Internet protocols TCP, UDP; TCP headers; format; connection management; transmission policy, flow control and congestion control; timer management
    • Socket programming for TCP and UDP

    Higher Protocol Layers

    • Domain Name System; other applications (WWW and Email);

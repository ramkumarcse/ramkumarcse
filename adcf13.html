Assignment 1

Write a simple TCP client and server. Both the client and server will run on the same machine. The client sends a succinct greeting to the server and server responds. The server will be invoked with the command
>> server serverport
and the client with the command
>> client serverIP serverport
For the server, choose a port number between greater than 4000. For the client determine the IP address of the server (which is also the IP address of the client!) and pass the argument serverIP as a character string in the dotted IP notation. (Complete By 16 Sep)

Assignment 2

Repeat Assignment 1 with UDP client and server. (Complete By 23 Sep)

Assignment 3

Write a simple TCP client that fetches HTML pages from web servers.
The client will be invoked with the command (for example)
>> webclient www.yahoo.com/
>> webclient www.msstate.edu/academics/
The client should get the response from the HTTP server and print it on the screen.

Hints:
Parse the string into two parts one before the first / and the rest
For example for
www.cse.msstate.edu/~ramkumar/cs4_6153.html
the first part is
www.cse.msstate.edu
and the second part is
/~ramkumar/cs4_6153.html
Now create a TCP socket,
get the IP address using gethostbyname() with the first part,
Establish connection with port 80
Send a HTTP request as contents of your data
For example, to fetch the page /~ramkumar/cs4_6153.html the request should be
GET /~ramkumar/cs4_6153.html HTTP/1.1\r\n Host: www.cse.msstate.edu\r\n \r\n \r\n
(Complete By October 14)
